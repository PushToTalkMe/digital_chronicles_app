# Digital Chronicles frontend part

## Команды

### Перед первым запуском

```bash
npm i
```

### Запустить в режиме `development`

```bash
npm run dev
```

### Собрать файлы для production (в папку `dist`)

```bash
npm run build
```

### Запустить сервер под файлы из директории `dist`

```bash
npm run preview
```
